二、程序运行过程中出现的问题总结

1、运行报数据库错误，然后进行数据库数据库相关配置。

参考网址：https://segmentfault.com/a/1190000004695637

2、小程序显示，不能访问IP，必须进行域名绑定。

阿里云进行域名添加:shop.yhjnl.com

小程序也要进行域名添加：域名需要进行ICP备案，备案搞了好长时间。

公众号添加域名，需要在服务器端，建立一个验证服务器。这样在网页提交之后，微信后台会发一个验证消息，到所在的域名服务器上面，服务端的token需要和微信上面填写的token保持一致。

参考：https://www.jianshu.com/p/a60579cc8b3a

3、beego程序运行起来之后，一直访问不了，但是相同端口，用c写的服务端，可以正常访问。

解决方法:把监听的ip设置为空。

beego.BConfig.WebConfig.Session.SessionOn = true
beego.BConfig.Listen.ListenTCP4 = true
beego.BConfig.Listen.HTTPAddr = ""
beego.BConfig.Listen.HTTPPort = 8080

beego.BConfig.Listen.EnableAdmin = true
beego.BConfig.Listen.AdminAddr = ""
beego.BConfig.Listen.AdminPort = 8088

程序终于运行起来了
